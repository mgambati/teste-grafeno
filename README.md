## Como Rodar o Projeto

No root do projeto:

```bash
# instala depências
yarn install

# Inicia server de dev
yarn start
```

Inicia o app em modo de desenvolvimento.<br />
Abra [http://localhost:3000](http://localhost:3000) para visualizar no Browser.
